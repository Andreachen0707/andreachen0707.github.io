LeetCode 105.Construct Binary Tree from Preorder and Inorder Traversal

Dec 2, 2018

The same method applies to LeetCode 106. Construct Binary Tree from Inorder and Postorder Traversal.

These two questions are very good to understand the structure of binary tree inorder, preorder and postorder traversal.

Keypoint to this problem I think is:

If using preorder, stack pop the first element as root and followed by left and right nodes. If using postorder, stack pop the last element as root and followed by right and left. The rest will be easy to implement using recursive.
